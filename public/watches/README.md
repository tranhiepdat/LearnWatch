# Thư mục ảnh đồng hồ

Thả ảnh thật vào đây để app **tự động** thay hình minh hoạ (SVG) bằng ảnh.

## Quy tắc

- Đặt tên file **đúng bằng `id` của mẫu** + đuôi `.jpg`. Ví dụ: `rolex-gmt-pepsi.jpg`.
- Ảnh nên là hình vuông (1:1), nền sạch, tối thiểu 600×600px.
- Không cần sửa code: có file đúng tên là hiện ngay; thiếu file thì app vẽ hình minh hoạ.

> Lấy ảnh ở đâu cho hợp pháp? Xem `docs/IMAGES.md`.
